# AutoCare

AutoCare — full-stack веб-приложение для сервиса подбора, диагностики и сопровождения автомобилей. Проект объединяет публичный сайт, личный кабинет клиента и административную панель в одном интерфейсе и одной кодовой базе.

## О проекте

Система построена как `pnpm`-монорепа и состоит из трех частей:

- `client` — frontend на React и Vite;
- `server` — backend API на NestJS и Prisma;
- `shared` — общие контракты, типы и схемы валидации, которые используются и на клиенте, и на сервере.

Такой подход позволяет держать бизнес-логику, API-структуры и типизацию синхронизированными без дублирования.

## Что есть в проекте

- публичные страницы с описанием сервиса и каталогом услуг;
- регистрация и авторизация пользователей;
- профиль клиента;
- управление автомобилями в личном кабинете;
- создание, просмотр и сопровождение заявок;
- отзывы по завершенным услугам;
- административный раздел для работы с услугами, отзывами и заявками.

## Основные сущности

- `User` — пользователь системы с ролями `USER` и `ADMIN`;
- `Car` — автомобиль пользователя;
- `CarSnapshot` — снимок автомобиля на момент оформления заявки;
- `Service` — услуга с категорией, стоимостью, slug и подробными блоками описания;
- `Order` — заявка на услугу;
- `Review` — отзыв по выполненной заявке;
- `Token` — refresh token для пользовательской сессии.

## Архитектура

Frontend организован по слоям `app`, `pages`, `widgets`, `features`, `entities`, `shared`.  
Backend разделен на доменные модули: `auth`, `cars`, `orders`, `profile`, `reviews`, `services`, `tokens`.

Проект рассчитан на сценарий, где пользователь может выбрать услугу, авторизоваться, добавить автомобиль, отправить заявку, отслеживать историю обращений и оставлять отзывы. Администратор при этом управляет содержимым каталога и рабочими данными через отдельный интерфейс.

## Идея проекта

AutoCare — это не просто витрина услуг, а единая рабочая среда для автомобильного сервиса. Пользователь получает понятный путь от выбора услуги до результата, а команда сервиса — инструменты для управления контентом, заявками и обратной связью.
